Professional Trading Dashboard

Real-time stock, crypto, forex, and commodity tracking desktop application built with PyQt5. Perfect for traders, investors, and financial professionals who need always-on-top market data.

Stock Ticker Pro is a professional desktop application that provides real-time financial data tracking with an always-on-top overlay. Built with PyQt5 and powered by Yahoo Finance API, it supports stocks, cryptocurrencies, forex pairs, commodities, and ETFs with clean mode for distraction-free monitoring.

Key Features

Real-Time Data
Live market data for stocks, crypto, forex, and commodities with automatic refresh intervals.
Always-On-Top
Keep your ticker visible above all other windows for constant market monitoring.
Clean Mode
Distraction-free minimal interface perfect for professional trading environments.
Customizable
Add your own symbols, adjust refresh rates, and customize display preferences.
Multi-Asset Support
Track stocks, cryptocurrencies, forex pairs, commodities, and ETFs all in one place.
Desktop Native
Built with PyQt5 for optimal performance and native Windows integration.
